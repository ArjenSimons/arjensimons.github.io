Bubbles Unite!
Project Type Group project | 3 students
Project Timeline 16 weeks | 2017-2018
Software Used Processing
Languages Used Java

Bubbles Unite! is a bubble shooter in which you must mix colours to shoot a colour of your choosing. You can only shoot in a vertical line while moving horizontally to evade obstacles and retrieve power-ups. Bubbles Unite! is the first group project during my game development education.
